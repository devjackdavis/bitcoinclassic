Sample configuration files for:
```

Upstart: bitcoinclassicd.conf
OpenRC:  bitcoinclassicd.openrc
         bitcoinclassicd.openrcconf
CentOS:  bitcoinclassicd.init
macOS:    org.bitcoinclassic.bitcoinclassicd.plist
```
